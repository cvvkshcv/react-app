## Interview question at wmvare

### Convert this object

```
{
   name: 'Test',
   email: 'test',
   address: {
      street: 'street name',
      pincode: '12345',
      city: 'Bangalore'
   }
}
```

### to this format

```
{
   name: 'Test',
   email: 'test',
   address_street: 'street name',
   address_pincode: '12345',
   address_city: 'Bangalore',
}

```
